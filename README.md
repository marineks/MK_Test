<li> %p</li>

## Install

Please clone the repo in the root folder of your project:

```sh
		|--- your files
---- ft_printf  |
		|--- mk_test
```
```sh
git clone https://github.com/marineks/MK_Test.git
```

## Usage

```sh
make
make tests
```
